# Cypress 3.0

Este projeto contém testes end-to-end automatizados utilizando **Cypress** no site da **Qazando**.

## Como usar

### 1. Instalar dependências
```bash
npm install
```

### 2. Abrir o Cypress
```bash
npx cypress open
```

### 3. Executar os testes em modo headless
```bash
npx cypress run
